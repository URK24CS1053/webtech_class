Introduction to Webtechnology
 - responsiveness: usable on all forms of devices
 - internet: networks of networks 
 - web technology is used to develop 
 - websites are static pages while web applications are interactive applications hosted on the web
Basic components:
 - frontend: html, css, js
 - backend: server, db
 - communication over the intenet through application layer protocols like http
Functionality of a website:
  - client->server->database->response
  - stack: html, css, js, Node.js or PHP, database like SQL or MongoDB
  - frameworks and APIs used
HTML
 - develop the structure and content of the webpage
 - html tags are also called html elements
 - HTML and HTTP protocols devveloped by T.B Lee at CERN.
 - W3C develops the web standards.
 - not recommended for styling, we use CSS for that

 #HTML Syntax
  - DOCTYPE tells the browser that we are using html 5
  - head: for the browser
  - meta tag contains information about the the webpage
  - body tag is for contents visible to the user like text(headings, paragraphs), links, tables, forms, iframes, multimedia
  - <!-->: this is a comment
  - some of the tags for html do not need to be closed like <br> for line break or <hr> for horizontal line
